fit parametric survival models using rstan
==========================================

- Duckweed lifespan data from: [Barks and Laird (2015)] (http://datadryad.org/resource/doi:10.5061/dryad.t938n)
- Models described in: [Pletcher et al. (2000)] (http://biomedgerontology.oxfordjournals.org/content/55/8/B381)
- Function to calculate WAIC from: [Vehtari and Gelman (2014)] (http://www.stat.columbia.edu/~gelman/research/unpublished/waic_stan.pdf)
